안녕하세요, 저는요!

깔끔하고 편안한 UI/UX을 보여주는 것이 기본자세라고 생각합니다. 쉽게 포기하지 않고 다른 분들의 경험과 기본 지식을 통해서 부족한 부분을 배우고 문제를 해결하기 위해 노력합니다.

계획

  • JavaScript 숙련도 올리기
  • 리액트 학습
Flex, Grid 사용 시맨틱 마크업 JS 기능 구현 Router 구현

기술

  • HTML

    HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용하면서 전체 레이아웃을 설계합니다.

  • CSS

    다양한 방식을 꾸준히 공부 중이며, 새롭게 지원하는 반응형, flex, grid, root 등을 포함하여 사용이 가능합니다.

  • SCSS

    제공하는 함수들을 활용하고 @mixin사용하면서 중복된 코드를 정리하고 효율적인 코드를 작성하기 위해 사용 중입니다. 현재는 CSS보다 더 선호하고 있습니다.

  • JAVASCRIPT

    가능한 플러그인을 사용하지 않고 기능 구현 위해 작성하고 있으며, 부족한 부분은 제이쿼리를 사용하고 있습니다.
